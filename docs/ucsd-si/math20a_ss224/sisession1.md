# Exponentials
1. $a^{(x+y)} = a^xa^y$
2. $a^{(x-y)} = \frac{a^x}{a^y}$
3. $(a^x)^y = a^{xy}$
4. $a^0 = 1$
5. $(ab)^x = a^xb^x$
6. $\left(\frac{a}{b}\right)^x = \frac{a^x}{b^x}$
